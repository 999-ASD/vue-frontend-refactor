## 七、设计模式应用

### 7.1 中介者模式（Mediator Pattern）

**应用场景**：Pinia Store 作为中介者

```
┌────────────────────────────────────────────────────────┐
│                    Mediator: useHospitalStore          │
├────────────────────────────────────────────────────────┤
│  职责：统一管理各模块间的通信，避免模块间直接耦合        │
└────────────────────────────────────────────────────────┘
              ▲               ▲               ▲
              │               │               │
   ┌──────────┴──────────┐ ┌──┴──────────┐ ┌──┴──────────┐
   │    Registration     │ │    Doctor   │ │   Pharmacy  │
   │      Module         │ │    Module   │ │   Module    │
   └─────────────────────┘ └─────────────┘ └─────────────┘
        │                        │                  │
        └────────────────────────┼──────────────────┘
                                 │
                          通过 Store 通信
                          (不直接互相调用)
```

**优势**：
- 降低模块间耦合度
- 提高代码可维护性
- 便于单元测试

### 7.2 工厂模式（Factory Pattern）

**应用场景**：DynamicForm 动态表单组件

```
┌────────────────────────────────────────────────────────┐
│                   Factory: DynamicForm.vue             │
├────────────────────────────────────────────────────────┤
│  职责：根据配置对象动态创建不同类型的表单字段            │
└────────────────────────────────────────────────────────┘
              │
              ▼
   ┌─────────────────────────────────────┐
   │         配置对象 (Config)           │
   │  { type: 'input', label: '姓名' }   │
   │  { type: 'select', label: '科室' }  │
   │  { type: 'number', label: '年龄' }  │
   └─────────────────────────────────────┘
              │
              ▼
   ┌─────────────────────────────────────┐
   │         产品族 (Products)           │
   │  ElInput  |  ElSelect  |  ElNumber  │
   │  ElDatePicker  |  ElTextarea        │
   └─────────────────────────────────────┘
```

**优势**：
- 扩展性强，新增字段类型只需添加配置
- 代码复用率高
- 符合开闭原则

## 八、技术亮点总结

| 维度 | 技术实现 | 设计意图 |
|------|----------|----------|
| **架构设计** | Vue 3 Composition API + Pinia | 响应式状态管理，代码复用 |
| **设计模式** | 中介者模式 + 工厂模式 | 解耦模块，提高扩展性 |
| **路由控制** | Vue Router + 路由守卫 | 统一认证管理 |
| **类型安全** | TypeScript 泛型 + 接口 | 编译期类型检查 |
| **持久化** | localStorage + Pinia 插件 | 数据持久化，刷新不丢失 |
| **组件封装** | 公共组件库 (CommonTable, DynamicForm) | 减少重复代码 |
| **构建工具** | Vite 5 | 快速热更新，按需加载 |
